# ✊ Rock Paper Scissors Game

A simple **Rock Paper Scissors web game** built with **HTML, CSS, and JavaScript**.
Play against the computer, track your score, and even enable **Auto Play mode**.

## 🎮 Features

* Play **Rock, Paper, Scissors** against the computer
* **Score tracking** (Wins, Losses, Ties)
* **LocalStorage** saves the score even after refreshing the page
* **Reset score** with confirmation message
* **Auto Play mode** (computer plays automatically)
* **Keyboard controls**
* Responsive design for smaller screens

## ⌨️ Keyboard Controls

* **R** → Rock
* **P** → Paper
* **S** → Scissors
* **A** → Toggle Auto Play
* **Backspace** → Reset Score

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts

## 🚀 How to Play

1. Open the **home page**.
2. Click **Play Now**.
3. Choose **Rock, Paper, or Scissors**.
4. The computer will randomly pick a move.
5. The result and score will update automatically.

## 🔄 Reset Score

Click **Reset Score** and confirm to clear all stored results.
